To make sure you can run this project, download Node.js and make sure the version is not 17.something 
because expo does not work properly with it. Next, download expo and instructions are in this link https://docs.expo.dev/get-started/installation/.

Now once you have the project up and running, download these libraries from react: 
https://reactnavigation.org/docs/bottom-tab-navigator/
https://github.com/oblador/react-native-vector-icons

Now, type in npm start into the terminal but make sure your current directory is in the project.
You can scan the barcode provided in the terminal to run the app and type ctrl+c to stop the app from running.
